# Product Requirements Document (PRD)

## 1. Product Vision
Build an AI-native, fully automated content production system
for high-CPM finance & economics YouTube channels,
optimized for retention, scalability, and profit.

## 2. Immutable Objectives
- Profit-first design
- Automation before human labor
- Data-driven decision making
- Agent-based modular architecture
- Scalable to multi-channel and SaaS

## 3. Success Metrics (North Stars)
- Average View Duration (AVD)
- Retention Curve
- CTR (Thumbnail + Title)
- RPM / CPM
- Time from idea â†’ published video

## 4. Explicit Non-Goals
- Emotion-driven content
- Manual-heavy workflows
- Trend chasing without monetization
- Non-repeatable creative processes

## 5. Core Deliverables
- End-to-end automated pipeline
- Agent orchestration system
- Continuous evaluation & rewrite loops
- Multi-channel scaling support
